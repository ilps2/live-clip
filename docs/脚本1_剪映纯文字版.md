## 剪映操作建议

- **总时长**：控制在 50-55 秒
- **转场**：全部用"淡入淡出"或直接硬切，不要花哨转场
- **字体**：思源黑体 Bold，数字用更大字号
- **关键帧**：每个"弹出"场景：开头缩放 50% → 0.3 秒后 120% → 0.2 秒后 100%
- **BGM**：开头 2 秒空白呼吸 → 渐入音乐 → 最后 3 秒音乐渐出
- **封面**：黑底+大字"我把视频降到了80×60像素，AI反而更准了？"
